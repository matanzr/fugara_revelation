שלומות!
הגעת לעמוד העלאת אנימציות עבור המיצב ״התגלות שמימית״ אשר יוצב בפסטיבל האור בירושלים 2018
בכדי שהתהליך יהיה קל ומגניב לנו ולכם יש מספר כללים פשוטים שצריך לעקוב אחריהם:
  1. העלו קבצי png בלבד
  2. סדרו את הקבצים כך שהקובץ הראשון הוא הפריים הראשון וכן הלאה. אין חשיבות לשמות שלהם רק שיהיו מסודרים.
  3. לדוגמא: a.png, b.png, ....
  4. הרזולוציה המומלצת לתמונות היא 400X400 ובכל מקרה גובה ורוחב שווים
  5. אין לחרוג מגודל של 40MB סה״כ
  6. אם יש בעיה לא לשכוח לנשום
אופן השימוש בעמוד:
  1. רשמו את שם האנימציה שלכם (תזכרו אותו, לשם אתם מעלים את הקבצים)
  2. בחרו את מספר המאוורר שאליו אתם מעלים כעת (אם יש לכם אנימציה אחת עבור כל המאווררים תעלו רק למאוורר 1)
  3. בחרו את הקבצים
  4. בחרו את הקבצים, חכו, זה יקח קצת זמן :)
  5. לחצו על upload to server, חכו, זה יקח קצת זמן :)
