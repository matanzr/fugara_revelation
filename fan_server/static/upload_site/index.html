toi.lab Fan site

Here you can upload new animations to the fan's database

All you need to do is follow this simple rules:
  1. Upload only .png files
  2. The fan reads in alphabetical order (same as they appear in you folder)
  3. for example: a001.png a002.png etc...
  4. Use square images. Recommended size for an image in a sequence is 400x400 pixels
  5. Breath!
To upload:
  1. Write the animation name (this is how you'll find it later)
  2. Select sequence files
  3. Wait for the processing to finish (it may take a few seconds. be patient)
  4. Once processing is done - You're ready to upload to fan

 

Meaningful names help to find it better later!
